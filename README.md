# DocumentAnalysis



<h2>Description</h2>
Project consists of a Python script allows you to scan a document to analyse speech.  
It will print: 
Length of Corpus
Number of Tokens Found
Vocabulary Size
Word Occurrence
Generate a Concordance
Word Index
Vocabulary

<h2>Language Used</h2>

- <b>Python</b> 


<h2>Environments Used </h2>

- <b>Windows 10</b>
- <b>Windows Server 2019 Datacenter</b>
